# 케이스 06: 타이틀 (Page Title / Card Title)

> 이 문서는 강제 규칙이 아닌 참고용 레퍼런스다.
> AI가 화면을 구성할 때 타이틀 영역의 구조를 판단하는 데 활용한다.

## 두 가지 타이틀 타입

Figma에 정의된 타이틀 컴포넌트는 `Type` 속성으로 구분된다.

### PageTitle

페이지 단위로 import 되는 화면의 최상단 타이틀.

```
┌─────────────────────────────────────────────────────────┐
│ 타이틀(xl/SemiBold)  [기능버튼]              [옵션 영역] │
│ 서브타이틀 및 디스크립션(sm/Normal, text-muted)          │
├─────────────────────────────────────────────────────────┤
│ 페이지 본문 콘텐츠                                      │
```

| 요소 | 스타일 |
|------|--------|
| 타이틀 텍스트 | `size-xl` + `font-semibold` + `text-default` |
| 기능 버튼 (선택) | 타이틀 옆에 배치, 보통 `secondary` 또는 `ghost` 스타일의 소형 버튼 |
| 옵션 영역 (선택) | 우측 끝 정렬, 필터/검색 등 기능 옵션이 들어갈 수 있음 |
| 디스크립션 (선택) | `size-sm` + `font-normal` + `text-muted` |

**특징:**
- 타이틀 행과 디스크립션 행으로 구성
- 타이틀 행 안에 기능 버튼 슬롯과 옵션 슬롯이 있음
- 디스크립션은 있을 수도, 없을 수도 있음
- 한 페이지에 PageTitle은 1개

### CardTitle

페이지 안에서 카드/섹션으로 구성되는 콘텐츠의 타이틀.

```
┌─────────────────────────────────────┐
│ 섹션 타이틀(md/SemiBold)            │
│ 카드 콘텐츠                         │
└─────────────────────────────────────┘
```

| 요소 | 스타일 |
|------|--------|
| 타이틀 텍스트 | `size-md` + `font-semibold` + `text-default` |

**특징:**
- PageTitle보다 작은 크기 (`size-md` vs `size-xl`)
- 디스크립션 행 없음
- 한 페이지에 여러 개 존재 가능

## 사용 판단 기준

| 상황 | 타이틀 타입 |
|------|-------------|
| 라우팅되는 페이지의 최상단 | PageTitle |
| 설정 페이지의 "프로필 정보", "알림 설정" 같은 섹션 | CardTitle |
| 대시보드의 "최근 상담", "공지사항" 같은 카드 | CardTitle |
| 모달/다이얼로그 제목 | 별도 (DialogTitle 컴포넌트) |
| 사이드 패널의 제목 | 상황에 따라 PageTitle 또는 CardTitle |

## 위계 구조 예시

```
┌─────────────────────────────────────────────────────┐
│ 프로젝트 관리 (PageTitle)         [+ 새 프로젝트]   │
│ 프로젝트를 생성하고 관리합니다                       │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌─────────────────────┐  ┌─────────────────────┐  │
│  │ 진행 중 (CardTitle)  │  │ 완료 (CardTitle)    │  │
│  │ ...                  │  │ ...                 │  │
│  └─────────────────────┘  └─────────────────────┘  │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │ 최근 활동 (CardTitle)                        │   │
│  │ ...                                         │   │
│  └─────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
```

PageTitle과 CardTitle의 크기 차이(`xl` vs `md`)가 자연스러운 시각적 위계를 만든다.
